LittleBigPlanet Tribute Page


LittleBigPlanet (LBP) is a physics-based 2.5D puzzle platformer video game series on multiple Playstation consoles. LBP is developed by Media Molecule, a video game studio based in England. LBP was also published by Sony Computer Entertainment. LBP allows users to playthrough and create their own levels, however, LBP's in-game funcionalities are do deep they allow users to create levels that go beyond the scope of a platforming level such as racing, fighting, top-down perspectives, and many more.

Main series games

Console titles:
  • LittleBigPlanet (2008)
  • LittleBigPlanet 2 (2011)
  • LittleBigPlanet 3 (2014)
Handheld titles:
  • LittleBigPlanet (PlayStation Portable) (2009)
  • LittleBigPlanet PS Vita (2012)
Spinoff titles:
  • Sackboy's Prehistoric Moves (2009)
  • LittleBigPlanet Karting (2012)
  • Run Sackboy! Run! (2014)
  • LittleBigPlanet Hub (Unreleased)
  • Sackboy: A Big Adventure (2020)

Fun Fact:

This was the prototpe for the playable character in LittleBigPlanet, this character's name is "Yellow Head". Yellow Head would be redesigned several times to eventually "Sackboy", who is in the final version of LittleBigPlanet
